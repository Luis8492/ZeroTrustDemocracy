# municipal_modules

自治体ごとの議会議事録を扱うモジュール群です。

新しい自治体をサポートする場合は、以下のサブディレクトリにそれぞれファイルを追加します。

- `config`  
  各自治体固有の設定を記述した YAML ファイルを配置します。例: `setagaya.yaml`
- `fetchers`  
  議事録をダウンロードするクラスを実装します。`BaseMinuteFetcher` を継承し、検索条件の設定や URL の抽出、ファイルの保存処理を記述します。
- `parsers`  
  ダウンロードした議事録を構造化データに変換するパーサーを実装します。`BaseMinuteParser` を継承し、議事録テキストからメタデータや発言を抽出します。

上記 3 つの要素を揃えることで、新しい自治体の議事録取得と解析が可能になります。
